# Content Factory · AI Film Studio

A desktop app for automated production of cinematic short-form video —
from idea to a ready-to-publish cut.

> 🪟 Public showcase: what the project does and where it's heading.
> 🔒 Implementation, methods and source code are private.

## Implemented
- Turn-key video production (script → voiceover → frames → video → thumbnail)
- Voiceover studio (multiple takes, export)
- In-app image & video generation with live cost tracking
- Built-in video editor: timeline, word-by-word captions, transitions, music, auto-assembly
- SEO studio: titles, descriptions, tags, keyword ideas, video analysis
- Community replies: YouTube · Instagram · Facebook
- Marketplace product-card generator
- Ad Studio: short ad videos for businesses + lead finder
- Multi-user mode with project ownership

## In progress / planned
